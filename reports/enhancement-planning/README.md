# Enhancement Planning Reports

This directory stores reusable static Phase 5 enhancement planning packets generated by:

```bash
python3 scripts/codex-task enhancement phase5-plan \
  --report-file reports/enhancement-planning/latest.json \
  --runbook-file reports/enhancement-planning/latest.md
```

The packet composes existing repository evidence into candidate rows for compaction continuity, discovery/search, AI-assisted template generation guardrails, optimization, optional MCP evaluation, metrics, and scanner-roadmap refreshes.

## Outputs

- `latest.json` - machine-readable candidate list with readiness, priority, evidence paths, missing evidence, refresh commands, next actions, and non-goal boundaries.
- `latest.md` - Markdown planning packet for review and handoff.

Task-local evidence should still be captured under the active work-tracking folder for the task that generated it.

## Boundaries

This report is planning-only. It does not install automatic triggers, enable optional MCPs, invoke external AI models, generate templates, apply performance changes, mutate Taskmaster from candidate rows, or contact external systems.
