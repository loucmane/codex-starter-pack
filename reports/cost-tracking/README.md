# Cost Tracking Reports

Generated by:

```bash
python3 scripts/template-cost-report
```

The report reads `templates/metadata/template-cost-policy.json`, optionally reads a usage JSON file, and writes:

- `latest.md`
- `latest.json`

Use `--strict` when fail-level budget outcomes should make the command exit nonzero. Missing usage data is reported as `not-measured` so the repository does not claim live billing telemetry it does not have.
