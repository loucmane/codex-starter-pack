# Success Metrics Reports

This directory is reserved for static success metrics dashboard packets generated by:

```bash
python3 scripts/codex-task success metrics \
  --report-file reports/success-metrics/latest.json \
  --runbook-file reports/success-metrics/latest.md
```

The packet composes existing repository evidence into a transparent success score across Taskmaster health, workflow state, template metrics, migration health, performance, final validation, and knowledge transfer.

It is intentionally file-backed and non-destructive. It does not create a live web dashboard, database, scheduler, alert, predictive service, notification, or external integration.
