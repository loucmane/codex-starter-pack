# Template Quality Reports

Template quality scorecards are static JSON/Markdown artifacts generated by:

```bash
python3 scripts/codex-task template quality-score \
  --report-file reports/template-quality/latest.json \
  --runbook-file reports/template-quality/latest.md
```

Use this report when an operator needs a transparent template quality score before changing shared templates, registry records, workflow guidance, or portable foundation assets.

The scorecard includes:

- weighted domain scores and grade
- metadata/drift evidence
- registry health evidence
- scanner complexity evidence
- performance evidence
- usage analytics evidence
- security audit evidence
- workflow continuity evidence
- quality gates
- improvement suggestions
- refresh commands
- explicit non-goals

This report is static and review-only. It does not install dashboards, mutate templates, enforce CI gates, run benchmarks, send notifications, train predictive models, or contact external services.
