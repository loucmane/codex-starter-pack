# Template Performance Reports

Generated by:

```bash
python3 scripts/template-performance-harness
```

The harness reads `templates/metadata/template-performance-policy.json`, measures selected portable foundation operations, and writes:

- `latest.md`
- `latest.json`

Use `--strict` when a fail-level performance outcome should make the command exit nonzero. Use `--baseline-file` to compare against a prior report or baseline and classify regressions.
