# Celebration Planning Reports

This directory stores reusable static celebration/readout planning packets generated by:

```bash
python3 scripts/codex-task celebration plan \
  --report-file reports/celebration-planning/latest.json \
  --runbook-file reports/celebration-planning/latest.md
```

The packet composes existing repository evidence into review-ready celebration material: achievement highlights, announcement draft sections, event/readout agenda items, demo candidates, recognition prompts, retrospective prompts, roadmap talking points, manual next steps, refresh commands, and explicit non-goals.

## Outputs

- `latest.json` - machine-readable celebration planning packet with evidence domains, readiness, draft materials, and manual follow-up steps.
- `latest.md` - Markdown packet for human review before external sharing.

Task-local evidence should still be captured under the active work-tracking folder for the task that generated it.

## Boundaries

This report is planning-only. It does not schedule meetings, send announcements, publish posts, create awards, generate slide decks, collect feedback, mutate Taskmaster from packet rows, or contact external systems.
