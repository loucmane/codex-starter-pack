- 2026-05-16 — [S:20260516|W:task109-foundation-installer-mcp|H:schemas/aegis|E:schemas/aegis/foundation-manifest.schema.json;schemas/aegis/profile.schema.json;schemas/aegis/install-plan.schema.json] Nullable repo paths in JSON Schema should use `anyOf` with `null` or a repo-path `$ref`; combining `type: [string, null]` with `not: {pattern: "^/"}` can accidentally reject `null` because string-only keywords are ignored for null before `not` is applied.
- 2026-05-16 — [S:20260516|W:task109-foundation-installer-mcp|H:scripts/_aegis_installer.py|E:tests/meta_workflow_guard/test_aegis_installer.py] Claude hook installation cannot copy only the headline gate scripts.
