# Task 236 Aegis Usability Convergence Roadmap

Task 236 is a planning-only convergence task. It does not authorize Capsule PR-4 retirement or
runtime behavior changes.

The current consumer evidence supports advisory passive recording but exposes four prerequisites
before legacy retirement: truthful mode-aware agent guidance, bounded command output, measured
worktree/subagent capture with reliable attribution, and a quiet deterministic witness at the
delivery boundary. The installer managed-update slice also needs incremental decomposition after
those contracts stabilize.

Tasks 237-244 implement those lanes. Task 244 owns the newly reproduced self-hosting gap where an
upstream source checkout cannot archive its final tracker without installed target state. Task 210
depends on Task 243, which transitively requires the complete convergence program and must issue
row-by-row parity decisions with proof, dogfood, and rollback. It is valid for Task 243 to conclude
that no surface is ready to retire.
